# <Prework Study Guide Webpage>

## Description

This Prework Study Guide was created to help coding bootcamp students reference and review topics learned during the prework. It is guide for those completing the prework as well as a reference for those that have already completed it. It covers HTML, CSS, Git and JavaScript.

## Installation

N/A

## Usage

Each section provides notes on different topics. For example, the HTML section contains notes on HTML. Other topics covered are CSS, Git and JavaScript. 

If you want a suggestion on what to study first you can open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.
